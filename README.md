## Overview

USOR (United States Oil Reserve) is a high-performance digital asset protocol dedicated to the conceptual tokenization of strategic energy reserves. By bridging traditional macro-economic energy indicators with decentralized finance, USOR establishes a transparent, data-driven "Strategic Digital Oil Pulse" on the Solana blockchain.

Built for the next era of American energy sovereignty, USOR leverages institutional-grade telemetry and hyper-scale liquidity pathing via the Jupiter Aggregator.

## Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn
- Solana CLI (for program deployment)

### 1. Clone & Initialize

```bash
git clone https://github.com/United-States-Oil-Reserve/USOR.git
cd USOR
npm run install:all
```

### 2. Launch Development Server

```bash
npm run dev
```

The dashboard will be available at `http://localhost:5173`

### 3. Build for Production

```bash
cd frontend
npm run build
```
